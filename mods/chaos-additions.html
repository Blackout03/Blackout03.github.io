Chaos Additions is an addon for the minecraft mod Chaos Awakens. Chaos Additions adds AIOTs for all Vanilla and Chaos Awakens Materials as well as its own material: Sapphire.

Latest Release:

- Added AIOT for Pure Chaos Meganium.
- Added Sapphire tool type.
- Added AIOT for new Sapphire tool type.
- Requires 0.9.0.3+



Archived Releases:

- Fixed bug where if mining some blocks like glass and glass panes the game would crash.
- Fixed bug where if mining some blocks like fossils the block would not drop.
- Requires 0.9.0.0+

- Adds AIOTs for all Vanilla and Chaos Awakens tool materials.
= An AIOT is a Pickaxe, Axe, Shovel and Hoe in One Tool.
